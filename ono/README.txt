--- HOW TO USE ---
1. "single_hist_cmpr.C" について
このスクリプトファイルは，一つのRUNの結果をTDCとADCに分けて表示します．
使用方法は，解析したいデータと同じディレクトリにこのスクリプトファイルを置き，
    root single_hist_cmpr.C
で実行してください．

2. "double_hist_arrng.C" について
このスクリプトファイルは，２つの異なるRUNの結果を
それぞれのRNUごとにTDCとADCの結果を表示します．
使用方法は，解析したいデータと同じディレクトリにこのスクリプトファイルを置き，
    root double_hist_arrng.C
で実行してください．

3. "double_hist_cmpr.C" について
このスクリプトファイルは，２つの異なるRUNの結果のTDCとADCを比較します．
使用方法は，解析したいデータと同じディレクトリにこのスクリプトファイルを置き，
    root double_hist_cmpr.C
で実行してください．
